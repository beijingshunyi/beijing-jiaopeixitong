### 技术实现

#### 前端
- 纯HTML + CSS + JavaScript
- 无需构建工具
- Chart.js 用于图表
- 响应式设计

#### 后端API调用
```javascript
// 获取学员列表
GET /api/staff/students

// 创建学员
POST /api/admin/users
{
  "name": "张三",
  "email": "zhangsan@example.com",
  "password": "password123",
  "role_id": 4,  // 学生角色
  "phone": "13800138000"
}

// 更新学员
PUT /api/admin/users/:id

// 删除学员
DELETE /api/admin/users/:id
```
